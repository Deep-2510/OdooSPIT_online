- **UI Library**: Material-UI (MUI) v5
- **State Management**: Context API + useReducer
- **Routing**: React Router v6
- **HTTP Client**: Axios
- **Charts**: Chart.js + React-Chartjs-2
- **Forms**: React Hook Form + Yup
- **Real-time**: WebSocket
- **Build Tool**: Vite
